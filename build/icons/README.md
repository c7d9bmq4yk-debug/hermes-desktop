# 应用图标

请在此目录放置应用图标文件：

## Windows 图标
- `icon.ico` - 256x256 像素的 ICO 格式图标

## macOS 图标  
- `icon.icns` - 1024x1024 像素的 ICNS 格式图标

## Linux 图标
- 放置 PNG 图标文件，尺寸包括：16x16, 32x32, 48x48, 64x64, 128x128, 256x256, 512x512

## 在线转换工具
- ICO: https://convertio.co/zh/png-ico/
- ICNS: https://cloudconvert.com/png-to-icns

注意：如果不放置图标，将使用 Electron 默认图标。
